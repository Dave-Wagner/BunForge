Class: Logger

Logger(options)

Logging Service Class. This class provides a unified interface for logging messages to different targets: - Console (default): Logs messages directly to the terminal. - CSV: Appends log messages in CSV format to a text file. - SQLite: Inserts log messages into a SQLite database for advanced querying using Bun's built-in API.
