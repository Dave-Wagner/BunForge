(constant) mergeObjects

Merge multiple objects deeply. Merges source objects into a target object recursively. For any key that exists in multiple objects, later sources override earlier ones.
Source:
  • utils/index.js, line 97
Example
const defaults = { a: 1, b: { c: 2 } };
const options = { b: { d: 3 } };
const merged = mergeObjects({}, defaults, options);
console.log(merged); // { a: 1, b: { c: 2, d: 3 } }
